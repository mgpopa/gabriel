Actuarial Reserving Playground

Personal

A personal project exploring how classical actuarial reserving methods can be made interactive and understandable through visualisation a simple web application.
This project demonstrates how core reserving techniques, such as Chain Ladder, Mack uncertainty, and Bootstrap simulation, can be explored visually using real claims triangles. The goal is to make traditionally opaque actuarial concepts easier to inspect, stress-test and explain.

  • Visualizes claims development triangles and how losses evolve over time
  • Implements classical reserving methods used in practice (Chain Ladder, Mack, Bootstrap)
  • Shows how uncertainty grows for immature accidents years and long-tailed lines
  • Allows scenario testing to explore how assumptions like inflation or tail factors affect reserves

PII-Safe Semantic Search for Insurance Claims

Personal

A personal project exploring how to enable semantic search and RAG over sensitive insurance claims notes without storing personally identifiable information (PII) in the vector index.

  • Unlocks semantic search on sensitive text while keeping PII out of vector stores
  • Reduces compliance risk by separating masked embeddings from governed raw text
  • Improves analyst productivity by enabling meaning-based search instead of exact keywords
  • De-risks vendor exposure by keeping vectors local and reusing existing governed data stores
